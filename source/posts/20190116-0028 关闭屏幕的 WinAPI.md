```c
#include <windows.h>
int main() {
  PostMessage(HWND_BROADCAST, WM_SYSCOMMAND, SC_MONITORPOWER, 2);
  return 0;
}
```

- 就是这么简单。纯 C，兼容 XP ~ 10，可在 PE 中使用。所以，为啥要去用那些带广告的，也许还需要 VCL 的关屏软件呢？
