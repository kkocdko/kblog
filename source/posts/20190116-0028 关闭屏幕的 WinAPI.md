```
title: 关闭屏幕的 WinAPI
date: 2019.01.16 00:28
tags: Code C
description: 与超时关屏效果相同
```

```c
#include <windows.h>
int main() {
    SendMessage(FindWindow(0, 0), WM_SYSCOMMAND, SC_MONITORPOWER, 2);
    return 0;
}
```

- 就是这么简单。纯 C，兼容 XP ~ 10，可在 PE 中使用。所以，为啥要去用那些带广告的，也许还需要 VCL 的关屏软件呢？
